# 🐍 Snake Game (Python)

A classic **Snake Game** built with **Python** using the Turtle graphics library.
This project demonstrates basic **Object-Oriented Programming (OOP)** concepts by separating the game logic into different modules.

---

## 🎮 Features

* Control the snake using **keyboard arrow keys**
* Food spawning system
* Score tracking system
* Snake grows when eating food
* Collision detection (walls & self)
* Modular Python project structure

---

## 🧠 Concepts Used

* Python Classes
* Object-Oriented Programming (OOP)
* Game loop
* Collision detection
* Turtle graphics

---

## 📂 Project Structure

```
snake-game/
│
├── main.py        # Main game loop
├── snake.py       # Snake class
├── food.py        # Food logic
├── scoreboard.py  # Score tracking
└── README.md
```

---

## ▶️ How to Run

1. Clone the repository

```
git clone https://github.com/your-username/snake-game.git
```

2. Open the project folder

```
cd snake-game
```

3. Run the game

```
python main.py
```
