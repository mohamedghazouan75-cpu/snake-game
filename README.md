# 🐍 Snake Game

A simple Snake Game built using Python and the Turtle graphics library.

## 🎮 Features
- Snake movement using keyboard arrows
- Score tracking system
- Clean object-oriented Python code
- Simple and fun gameplay

## 🛠️ Technologies Used
- Python
- Turtle Graphics

## ▶️ How to Run

1. Install Python
2. Download the project
3. Run:

```bash
python main.py
